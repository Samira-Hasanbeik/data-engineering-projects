# Sample Data

This folder contains a small subset of the original dataset used in this project.

## Description
The data consists of HTML snapshots of the front pages of multiple German news platforms.  
Each file represents a snapshot of a news website at a specific timestamp.

## Included Sources

- Tagesschau
- Heute
- Spiegel
- Welt
- Zeit
- Bild
- Frankfurter Allgemeine Zeitung (FAZ)
- Handelsblatt

## File Format
Format: `.html`

## Note
The full dataset is not included due to size limitations.
