# News Monitor Pipeline

ETL pipeline for analyzing multi-platform news data.

## Overview
This project processes HTML snapshots from multiple news platforms(bild, FAZ, Tagesscau, Spegel, Welt, Zeit, handelsblatt, Heute)  and transforms them into structured data for analysis.

## Features
- Data extraction from HTML
- Data cleaning and transformation
- PostgreSQL database design
- SQL-based analytics
- Text pattern detection using regex

## Tech Stack
- Python
- SQL
- PostgreSQL
- Regex

## Project Goal
To analyze how different news platforms report topics and how prominently they appear.

## Summary
Building a data pipeline for news data (HTML → Database → Analysis)
